what is Computer Science?

1. Computer Science란 컴퓨터를 사용하여 문제를 해결(Problem-Solving) 하는 방법을 연구하는 것
2. Programming을 통해 문제의 Solution을 구현하는 것

이때 programming은 Algorithm + Data Structuer이다. 즉 데이터를 갖고 알고리즘을 통해 결과값/결과물을 구현하는 것

what is Algorithm?

컴퓨터로 문제를 풀기 위한 단계적인 절차
입력(input)을 출력(output)으로 변환해주는 Computational steps
같은 문제를 서로 다른 방법으로 풀어 나가면서 어떤 방식이 효율적인지를 확인

위 세가지 모두 Algorithm의 의미이자 역활이다.

Algorithm의 표현방법은 크게 4가지로 나눌수 있다.
1. 영어나 한글과 같은 자연어
2. 흐름도(FLow chart)
3. 유사코드(Pseudo-code)
4. 특정 프로그래밍 언어(C++, Python, Java, Java Script 등등)

그리고 코딩은 이러한 4가지 방법중에서 프로그래밍 언어를 통해 알고르즘을 표현하는 것이다.
